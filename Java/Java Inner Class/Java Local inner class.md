A local inner class in Java is a class defined within a method or block, which can access the members of its enclosing class and local variables of the method (effectively final in JDK 1.8+).

## What is Local Inner Class in Java?

A local inner class is a class defined inside a method or a block (such as a for loop or an if statement). Unlike member inner classes, local inner classes are not members of the enclosing class; they exist only within the block where they are defined.

Local inner classes cannot have access modifiers like public, private, or protected, but they can be marked as final or abstract. They can access fields and methods of the enclosing class, including private members.

To use a local inner class, you must instantiate it within the same method or block where it is defined.

## How to Create a Local Inner Class?

A local inner class is created by defining a class inside a method or block. To create local inner class, first declare the outer class and a method or block where you want the inner class. After that, define the local inner class inside that method or block and implement its members. Finally, instantiate the local inner class within the same method to call its methods.

#### Rule: Local variables can't be private, public, or protected.

## Rules for Local Inner Class

The following rules are applied when creating a local inner class:

- **Access Modifiers:** Local inner classes cannot be declared as private**,** public, or protected.
- **Scope:** A local inner class cannot be accessed from outside the method in which it is defined.
- **Access to Local Variables:**
    - Before JDK 1.8, a local inner class could only access local variables that were declared final.
    - From JDK 1.8 onwards, local inner classes can access non-final local variables as well.
