A static nested class in Java is a class defined inside another class with the static modifier. It cannot access non-static (instance) members of the outer class directly, but it can access static members, including private static fields and methods.

A static nested class is associated with the outer class, so it can be accessed using the outer class name without creating an instance of the outer class.

## Creating a Static Nested Class

A static nested class is created by defining a class inside another class with the [static keyword](https://www.tpointtech.com/static-keyword-in-java). To use the static nested class, you can instantiate it using the outer class name without creating an object of the outer class.

### Syntax

The syntax to create a static nested class is as follows:

[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)

1. class OuterClass {  
2.     static int outerData = 50; // Static member of outer class  

3.     // Static nested class  
4.     static class StaticNestedClass {  
5.         void display() {  
6.             System.out.println("Outer class static data: " + outerData);  
7.         }  
8.     }  
9. }  

### Example

The following example demonstrates creating and using a static nested class:

[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)

1. class TestOuter1{  
2.   static int data=30;  
3.   static class Inner{  
4.    void msg(){System.out.println("data is "+data);}  
5.   }  
6.   public static void main(String args[]){  
7.   TestOuter1.Inner obj=new TestOuter1.Inner();  
8.   obj.msg();  
9.   }  
10. }  

**Output:**

data is 30

In this example, you need to create the instance of static nested class because it has instance method msg(). But you don't need to create the object of the Outer class because the nested class is static and static properties, methods, or classes can be accessed without an object.

## Static Nested Class with Static Method

A static nested class can have static members and methods. If a method inside the static nested class is static, you don't need to create an instance of the nested class to call it. You can access the static method directly using the outer class name followed by the nested class name.

### Example

The following example demonstrates creating static nested class with static method:

[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)[](https://www.tpointtech.com/static-nested-class#)

1. public class TestOuter2{  
2.   static int data=30;  
3.   static class Inner{  
4.    static void msg(){System.out.println("data is "+data);}  
5.   }  
6.   public static void main(String args[]){  
7.   TestOuter2.Inner.msg();//no need to create the instance of static nested class  
8.   }  
9. }  

**Output:**

data is 30
